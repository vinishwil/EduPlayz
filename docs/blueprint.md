# **App Name**: Eduplayz Adventures

## Core Features:

- Interactive Home Screen: Display animated characters and large buttons for learning modules.
- Alphabet Learning: Show alphabets with associated images and text-to-speech audio.
- Number Learning: Teach numbers with visual objects and spoken pronunciations.
- Find the Letter Game: Game to find the correct alphabet from multiple choices.
- Find the Number Game: Interactive game to identify the correct number.
- Speech Interpretation (Premium): Use AI tool (Gemini) to interpret and match unclear kid speech for the games as a premium feature. This enhances the voice recognition feature to provide more accurate feedback even with imperfect pronunciation.
- Progress Tracking: Track completed lessons and games using localStorage to display progress.
- Mini-Games: Implement at least 6–8 mini-games with reusable game engine logic and playful animations. Games should be short, rewarding, and accessible to kids aged 3–6.

## Style Guidelines:

- Primary color: Saturated and vibrant blue (#29ABE2) to create an inviting atmosphere reminiscent of the sky, and universally appealing to children.
- Background color: Very light blue (#E1F5FE) with low saturation (around 20%) to maintain focus on the learning content.
- Accent color: Analogous hue in the purple spectrum (#8E44AD) with higher brightness and saturation than the background, to highlight interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable feel.
- Use cartoon-style icons with bright colors to represent different modules and actions.
- Design with rounded cards and large buttons for easy interaction.
- Incorporate micro-animations on hover, click, and success events to provide engaging feedback.
